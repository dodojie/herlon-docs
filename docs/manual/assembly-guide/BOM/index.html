编号名称数量用途

4短铜柱4个连接主板与下外壳
5黄铜内六角螺丝4个固定上外壳

10中空亚克力外壳1片无墨水屏时，用作下外壳，有墨水屏时，用作上外壳
1113P排线1条用于连接主板与微雪2.9寸墨水屏
128P排线1条用于连接主板与大连佳显2.9寸墨水屏(不带触摸那款)
